## ✅ 已完成

### 1. 核心功能
- ✅ 批量 JSON 转 Excel 处理脚本
- ✅ 自动生成 3 种格式的 Excel 文件（汇总/详细/统计）

## 🚀 使用流程

### 完整工作流
```
1. 用 AnyToCopy 提取抖音视频文案
   ↓
2. 保存为 JSON 格式
   ↓
3. 放入 input/douyin_texts/ 目录
   ↓
4. 运行 ./run.sh
   ↓
5. 在 output/ 查看生成的 Excel
```

### JSON 格式
```json
{
  "视频链接": "https://v.douyin.com/xxxx/",
  "视频标题": "标题",
  "作者": "作者名",
  "视频时长": "04:22",
  "发布时间": "2026-03-23",
  "完整文案": "AnyToCopy 提取的完整内容..."
}
```

## 📝 注意事项

1. **输出目录**：`output/` 已在 `.gitignore` 中，不会被提交
2. **输入数据**：`input/douyin_texts/` 中的真实数据建议手动添加到 `.gitignore`
3. **编码**：所有 JSON 文件必须使用 UTF-8 编码
4. **依赖**：仅需 `openpyxl` 一个依赖
